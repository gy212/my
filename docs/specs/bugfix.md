# Bugfix Spec: 响应式自适应大小适配

## 缺陷摘要

像素风格作品集网站 `combined_portfolio.html` 在小屏设备（< 768px）上存在系统性布局问题：溢出、字体过大、元素重叠、固定尺寸不缩放。

## 影响范围

- **受影响页面**：`combined_portfolio.html`（唯一页面）
- **受影响断点**：主要影响 < 768px（手机），部分影响 768px-1024px（平板）
- **严重程度**：高 - 移动端基本不可用

| 822 | `text-[160px]` | 笑脸符号在手机上严重溢出 |

## 期望行为

1. 所有装饰性大字体在小屏上自动缩小或隐藏
2. 固定高度容器改为自适应高度
3. 绝对定位装饰元素在小屏上收敛或隐藏
4. 所有文字内容在 320px 宽度下无水平溢出
5. 整个页面在任何宽度下 `overflow-x` 无可见水平滚动条
6. 保持像素风格设计的视觉特征

## 必须保持不变的行为

1. 桌面端（> 1024px）的所有视觉效果和布局不变
2. ASCII Hero 动画效果不变
3. 自定义游标效果不变
4. 滚动进度条效果不变
5. Scroll Reveal 动画效果不变
6. 所有链接和交互功能不变
7. 颜色方案和像素风格美学不变
