KEYBOARD_TYPER // INTRO

给不能粘贴的场景补一个可控输入层

这是一个全中文界面的键盘模拟输入器，适合在线表单、考试系统、远程桌面或其它限制复制粘贴的环境。

核心价值不是“存文本”，而是把现成内容按键盘事件重新打进去，让被限制的输入框也能使用。

把图片内容转成可编辑文档，而不是只做截图留档

仓库里保留了桌面运行入口、配置目录和 UI 结构，目标是把“识别”直接推进到“可交付文档”。
